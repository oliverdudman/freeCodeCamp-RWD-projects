Introduction

Flexible box is a layout model introduced in CSS3 for the design of flexible responsive layouts. Flexboxes can be either a row or column and each flex container can only have one row or colum at any given time. Flexboxes have the follwing properties:

  • align-content
  • align-items
  • align-self
  • flex
  • flex-basis
  • flex-direction
  • flex-flow
  • flex-grow
  • flex-shrink
  • flex-wrap
  • justify-content
  • order

Getting started

In order to start using flexboxes, first set the display on the parent container to flex and set the flex-direction to either row or column (row is default): 


          .parent {
            display: flex;
            flex-direction: row;
          }

To center an element vertically and horizontally, set both the justify-content and align-items properties to center. In order to take effect, the width and height also need to be set. 


          .parent {
            display: flex;
            justify-content: center;
            align-items: center;
            flex-direction: row;

            width: 300px;
            height: 300px;
          }

A centered element

Flexbox axis

A flexbox has two axis: the main axis and the cross axis. The flex-direction property sets the main axis and the cross axis is perpendicular to the main axis.

Aligning content

The content of a flexbox can be aligned using the justify-content and align-items properties. The justify-content property aligns the content along the main axis and can have the following values:

  • center
  • flex-start
  • flex-end
  • space-between
  • space-around
  • space-evenly

            <style>
              .parent {
                display: flex;
                justify-content: ;// alignment option
                flex-direction: row;

                width: 100%;
                height: 100%;
              }
            </style>

            <body>
              <div class="parent">
                <div class="box"></div>
                <div class="box"></div>
                <div class="box"></div>
              </div>
            </body>

The align-items property aligns the content on the cross axis. When the flex direction is set to row, the align-items property will align the content on the vertical axis. 


            <style>
              .parent {
                display: flex;
                align-items: ;// alignment option
                flex-direction: row;

                width: 100%;
                height: 100%;
              }
            </style>

            <body>
              <div class="parent">
                <div class="box"></div>
                <div class="box"></div>
                <div class="box"></div>
              </div>
            </body>

Wrapping content

A flexbox only has one row or column, but by using the flex-wrap property, a row will span multiple lines if the width of the content exceeds the width of the flexbox. Flex-wrap can have the following values:

  • no-wrap (default)
  • wrap
  • wrap-reverse

          <style>
            .parent {
              display: flex;
              flex-direction: row;
              flex-wrap: ;// wrap option

              width: 100%;
              height: 100%;
            }
          </style>

          <body>
            <div class="parent">
              <div class="box">A</div>
              <div class="box">B</div>
              <div class="box">C</div>
              <div class="box">D</div>
              <div class="box">E</div>
              <div class="box">F</div>
              <div class="box">G</div>
            </div>
          </body>

When content is wrapped, the align-content property can be used to align content on the cross axis. Align-content takes the same values as justify-content and, for a flexbox with a flex-direction of row, will work on the vertical axis. 


          <style>
            .parent {
              display: flex;
              flex-direction: row;
              flex-wrap: wrap;
              align-content: ;// alignment option

              width: 100%;
              height: 100%;
            }
          </style>

          <body>
            <div class="parent">
              <div class="box">A</div>
              <div class="box">B</div>
              <div class="box">C</div>
              <div class="box">D</div>
              <div class="box">E</div>
              <div class="box">F</div>
              <div class="box">G</div>
            </div>
          </body>
